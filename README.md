# JobSpark — Secure Job Matching Platform

JobSpark is a full-stack job matching platform designed to support secure authentication, scalable backend services, and cross-platform applications.

## Architecture

- Backend: Golang (Fiber Framework)
- Frontend: Web Application
- Mobile: Mobile Application
